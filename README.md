# Iris Flower Classification using KNN

## Description
This project uses the Iris dataset and a K-Nearest Neighbors (KNN) classifier to classify flowers into three species.

## Features
- Data preprocessing
- Feature scaling
- KNN Classification
- Accuracy Evaluation
- Classification Report
- Confusion Matrix

## Technologies Used
- Python
- NumPy
- Scikit-learn

## Dataset
Iris Dataset from Scikit-learn

## How to Run

pip install -r requirements.txt

python iris_classification.py

## Results
The model classifies Iris flowers into:
- Setosa
- Versicolor
- Virginica
